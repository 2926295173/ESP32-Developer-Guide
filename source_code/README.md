## 源码备注

- hx-https-mbedtls：HTTPS获取天气

- hx-oled：I2C-OLED显示

- hx-pwm：PWM全彩LED灯显示

- hx-sc-http：HTTP获取城市温度
- hx-sc：SmartConfig一键配置
- hx-sht30：I2C-读取温湿度
- hx-tcp： ESP32的TCP连接
- hx-uart：两个UART实验
- hx-udp：ESP32的UDP广播
- hx-wifi： 新建一个WIFI热点
